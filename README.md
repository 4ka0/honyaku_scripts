# Project Title

Script to check whether correct terminology is being used in a translation.

## Running the script

From the terminal:
```
python3 term_checker.py translation.tmx glossary.txt
```

## Prerequisites

Built using the following:

```
python 3
colorama 0.4.3
translate-toolkit 2.5.0
pytest 5.4.1
```

## Tools etc.

* Visual Studio Code
* macOS 10.14.6
